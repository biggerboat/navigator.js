Backbone + Navigator State Map Example (depends on jquery, backbone and underscore)

Welcome to the third example of the Navigator-JS library. Added to this example are 4 elements. A red, green and blue square, and a black circle. By clicking the menu, you can change the navigation state. You can also type in a path in the debug console.

The main purpose of this example is to only instantiate Backbone-views the moment the Navigator decides they are needed.

Red Square | Green Square | Blue Square | Black Circle
